# Deeptalk

> A free, open source, privacy-first alternative to Discord.

## What is this?

Discord was good once. Then the money got involved. Today it sells user data, requires ID verification, and treats its users as statistic instead of people.

Deeptalk is being built to fix that. A communication platform with one goal — give people a better space that they actually own and control.

## Core values

- Privacy. No data selling. No ID verification. No surveillance.
- Everything is free or earnable. No paywalled profiles, no exclusive badges, no dark patterns.
- Open source from day one. The community owns this, not a me.
- Built for people. If you need a tutorial to send a message, the design has failed.

## Planned features

- Peer-to-peer mesh networking — users are the relays, no central server storing your conversations
- Plugin ecosystem — community built features, no bots, no attack vectors
- Full UI customization — drag everything, hide what you don't use, avatar-based themes
- No username wars — use any name, unique ID handles disambiguation
- Badge system where every badge is always earnable, no seasonal exclusives, ever
- Desktop app built in Tauri — Windows, Mac, Linux
- Cross platform mobile to follow

## Status

Early stage. Vision and architecture defined. Looking for developers, designers, and contributors who believe people deserve better.

## Want to help build this?

Open an issue, start a discussion, or reach out directly. All skill levels welcome — developers, designers, artists, anyone who gives a damn.
